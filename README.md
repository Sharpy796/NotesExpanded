# NotesExpanded
Cast your spells with expanded song!

This mod fills out the Kantele and Ocarina notes across several octaves: 2 for Kantele and 3 for Ocarina. If you want to find them, go look for the Ocarina!

Here's the link to the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3705021946) version of the mod!
